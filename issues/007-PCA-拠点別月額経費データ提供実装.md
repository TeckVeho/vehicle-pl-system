# PCA: 拠点別月額経費データの提供実装

**対象:** PCA 側エンジニア

## 1. 概要

拠点別月額経費（旅費交通費・消耗品・修繕費等 20 科目）をイズミクラウドが取得できる形式で提供する実装。データは PCA → イズミクラウド → IZUMI の流れで連携される。

### 達成目標

* [ ] 拠点別・勘定科目別の月額経費をイズミクラウドが取得できる形式で提供（API、ファイル、DB 連携など）
* [ ] 20 科目の科目コードとのマッピングが整備されていること

---
## 2. 実装仕様

### データエクスポート

| 項目 | 仕様 |
|------|------|
| 提供形式 | API、ファイル、DB 連携のいずれか |
| データ内容 | 拠点別・勘定科目別の月額経費 |
| 連携先 | イズミクラウドが取得し、`POST /api/location-monthly-expenses/sync` で IZUMI に送信 |

### 対象科目（20 科目）

旅費交通費(6150)、消耗品(6151)、修繕費(6154)、通信費(6156)、水道光熱費(6159)、保険料(6160)、租税公課(6162)、他手数料(6164)、交際接待費(6165)、会議費(6166)、広告宣伝費(6167)、諸会費(6168)、地代家賃(6171)、借家料(6172)、賃借料(6173)、保守料(6174)、図書研修費(6177)、減価償却費(6178)、雑費(6188)、事故賠償費(6189)

### 科目マッピング

PCA の勘定科目と IZUMI の科目コード（上記 20 科目）との対応を整備すること。イズミクラウドがマッピングを適用して送信する。
