# ITP: 燃費・道路使用料データの提供実装

**対象:** ITP 側エンジニア

## 1. 概要

車両別・月別の燃費と道路使用料をイズミクラウドが取得できる形式で提供する実装。データは ITP → イズミクラウド → IZUMI の流れで連携され、`vehicle-monthly-costs/sync` の各車両データに含めて送信される。

### 達成目標

* [ ] 車両別・月別の燃費（L）をイズミクラウドが取得できる形式で提供
* [ ] 車両別・月別の道路使用料（割引適用前の生データ）をイズミクラウドが取得できる形式で提供

---
## 2. 実装仕様

### 燃費データの提供

| 項目 | 仕様 |
|------|------|
| 提供形式 | API、ファイル、DB 連携のいずれか |
| データ内容 | 車両別・月別の燃費（L） |
| 連携先 | イズミクラウドが `vehicle-monthly-costs/sync` の各車両データに `fuelEfficiency` として含めて送信 |
| IZUMI での用途 | 燃料費（6175）の算出に使用 |

### 道路使用料データの提供

| 項目 | 仕様 |
|------|------|
| 提供形式 | API、ファイル、DB 連携のいずれか |
| データ内容 | 車両別・月別の道路使用料（割引適用前の生データ） |
| 連携先 | イズミクラウドが `vehicle-monthly-costs/sync` の各車両データに `roadUsageFee` として含めて送信 |
| IZUMI での用途 | 道路使用料（6176）の算出に使用（拠点別計算パラメータの割引率を適用） |

# ITP: Triển khai cung cấp dữ liệu nhiên liệu・phí đường bộ

**Đối tượng:** Kỹ sư phía ITP

## 1. Tổng quan

Triển khai cung cấp nhiên liệu và phí đường bộ theo xe・theo tháng dưới dạng Izumi Cloud có thể lấy được. Luồng: ITP → Izumi Cloud → IZUMI, đưa vào dữ liệu từng xe của `vehicle-monthly-costs/sync`.

### Mục tiêu đạt được

* [ ] Cung cấp nhiên liệu theo xe・theo tháng (L) dưới dạng Izumi Cloud có thể lấy
* [ ] Cung cấp phí đường bộ theo xe・theo tháng (dữ liệu gốc trước giảm giá) dưới dạng Izumi Cloud có thể lấy

---
## 2. Thông số kỹ thuật triển khai

### Cung cấp dữ liệu nhiên liệu

| Mục | Thông số |
|-----|----------|
| Định dạng cung cấp | Một trong API, file, liên kết DB |
| Nội dung dữ liệu | Nhiên liệu theo xe・theo tháng (L) |
| Đích liên kết | Izumi Cloud đưa vào dữ liệu từng xe của `vehicle-monthly-costs/sync` với trường `fuelEfficiency` |
| Dùng trên IZUMI | Tính chi phí nhiên liệu (6175) |

### Cung cấp dữ liệu phí đường bộ

| Mục | Thông số |
|-----|----------|
| Định dạng cung cấp | Một trong API, file, liên kết DB |
| Nội dung dữ liệu | Phí đường bộ theo xe・theo tháng (dữ liệu gốc trước giảm giá) |
| Đích liên kết | Izumi Cloud đưa vào dữ liệu từng xe của `vehicle-monthly-costs/sync` với trường `roadUsageFee` |
| Dùng trên IZUMI | Tính phí đường bộ (6176) (áp dụng tỷ lệ giảm từ tham số tính toán theo địa điểm) |
